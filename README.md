# chatapp_flutter

 This ia simple chat app built using flutter. Flutter is an open-source UI software development kit created by Google. It is used to develop cross platform applications for Android, iOS, Linux, Mac, Windows, Google Fuchsia, and the web from a single codebase. This was done with the help of the model given by [freecodecamp.](https://www.freecodecamp.org/news/build-a-chat-app-ui-with-flutter/)
